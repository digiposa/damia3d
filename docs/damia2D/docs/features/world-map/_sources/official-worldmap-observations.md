## Observations canon directement visibles sur la map

### Gloriano (nord-ouest, terrain neigeux + ruines glaces)

- **Zenebatos** ⚠️ confirmation canon position : **NORD de Gloriano** (extrême nord-ouest), pas Mille Seseau
- **Vellweb** : centre-ouest Gloriano (7 towers canon)
- **Snowfield** : centre-ouest Gloriano (passage)
- **Tower of Flanvel** : émerge du Kashua Glacier (cohérent canon "half-buried")
- **Kashua Glacier** : sépare Gloriano de Mille Seseau (cohérent canon)

### Mille Seseau (nord, terrain neigeux + forêts vertes)

- **Kadessa** ⚠️ : **extrême nord-est** Mille Seseau (cohérent "Divine Dragon flies above" canon — ruins Forbidden Land canon position confirmée)
- **Mountain of Mortal Dragon** : nord Mille Seseau (volcan rouge visible — cohérent "volcano north Mille Seseau where Divine Dragon sealed" canon)
- **Deningrad** : centre Mille Seseau (capital)
- **Forest of Winglies** : nord-est Mille Seseau (cohérent canon "Many surviving Winglies fled here")
- **Evergreen Forest** : centre Mille Seseau (cohérent canon "large forest central Mille Seseau")
- **Neet** : extrême est Mille Seseau (côte Illisa Bay — cohérent "close to Illisa Bay" canon)

### Zone centrale (entre Mille Seseau et Gloriano/Tiberoa)

- **Mayfil** : centre nord, entre Mille Seseau et Death Frontier (Death City)
- **Furni** ⚠️ : **sud Mille Seseau, côte ouest Illisa Bay** (Water City, position canon)
- **Death Frontier** : grand désert orange centre Endiness (ruines visibles + Home of Gigantos voisinage)
- **Home of Gigantos** : sud-est Death Frontier (voisinage Valley of Corrupted Gravity)
- **Ulara** : sud Death Frontier (Spring Breath Town, près petites ruines visibles)

### Tiberoa (centre-sud, terrain rougeâtre/désertique)

- **Donau** : nord Tiberoa (cohérent "northern coast Illisa Bay" canon)
- **Barrens** : centre Tiberoa (désert)
- **Valley of Corrupted Gravity** : centre-nord Tiberoa
- **Fletz** : sud Tiberoa (capital, Twin Castle visible — cohérent sud Aquaria Coral Reef canon)
- **Barrier Station** ⚠️ **NEW canon location** ! Visible entre Fletz et Valley of Corrupted Gravity. Non mentionné par wiki/fandom Endiness pages. À investiguer.

### Illisa Bay (océan centre, entre 3 countries)

- **Prison Island** : île dans Illisa Bay (centre)
- **Lidiera** : île est Illisa Bay
- **Undersea Cavern** : connecte îles canon
- **Fueno** : sur **Suncrest Island** (est Illisa Bay)

### Serdio (est, terrain temperate verdoyant)

- **Bale** : nord Serdio (capital Basil)
- **Limestone Cave** : est, près Bale
- **Hoax** : nord-centre Serdio
- **Marshland** : centre Serdio (cohérent humid forest canon)
- **Volcano Villude** : centre-sud Serdio (volcan visible)
- **Prairie** : centre-est Serdio
- **Seles** : extrême est Serdio (Dart's home town)
- **Forest** : voisinage Seles
- **Shrine of Shirley** : centre-sud Serdio
- **Nest of Dragons** ⚠️ confirme nom officiel canon (vs "Nest of Dragon" autres sources)
- **Lohan** : centre-sud Serdio
- **Kazas** : extrême sud Serdio (capital Sandora, Black Castle)
- **Hellena Prison** : sud-est extrême Serdio

### Broken Islands (ouest, îles séparées)

- **Aglis** : sud-ouest, île isolée (cohérent "underwater near Rouge" canon)
- **Rouge** : ouest, île (cohérent "eastern Broken Islands" canon)

## Pattern canon géographique

- **Gloriano** = nord-ouest (Vellweb + Snowfield + Zenebatos + Tower of Flanvel + Kashua Glacier)
- **Mille Seseau** = nord-est (Deningrad + Mountain of Mortal Dragon + Forest of Winglies + Kadessa + Neet + Evergreen Forest + Furni sud)
- **Mayfil** = entre Gloriano-Mille Seseau et Death Frontier (centre nord)
- **Death Frontier** = centre (Ulara + Home of Gigantos + ruines)
- **Tiberoa** = centre-sud (Fletz + Donau + Barrens + Valley + **Barrier Station NEW**)
- **Illisa Bay** = océan centre-est (Lidiera + Prison Island + Suncrest+Fueno + Undersea Cavern)
- **Serdio** = est (Bale + Kazas + Lohan + Seles + Hoax + Marshland + Volcano Villude + Prairie + Forest + Limestone Cave + Shrine of Shirley + Nest of Dragons + Hellena)
- **Broken Islands** = ouest (Aglis + Rouge)
- **Rudra Bay** = ouest sea (entre Broken Islands et mainland)
- **Queen Fury** route Disc 2 ship = Aglis/Rouge ↔ Fletz Tiberoa
