## Description ⭐

Dark Elf is a **Darkness-element monster** which is located within the **Evergreen Forest**.

It has the ability to **Petrify an ally, effectively removing a character from combat in one shot**. It sometimes appears together with the **Forest Runner, which can more or less do the same**. **Kill the Elf first**, since the **Forest Runner can only cast "Wooing" when at red health**.
