## Items overview

**Used items** are consumable resources that provide a variety of effects that are organized below into **Recovery, Random, Attack, Repeat, and Miscellaneous**. **Only 32 may be held at a time**, with attempts to open chests containing more resulting in the message, "**Cannot carry any more items.**" When defeating an Enemy that drops an item while already at the cap, the game gives the option to trade the item out before being discarded. Most items can be bought in item shops, but some are only attainable via treasure chests or from defeating enemies. While all of them can be used in battle, some items can also be used or discarded from the Pause Menu and may have different effects as explained in the tables below.

**Armed items** are permanent resources like armor, weapons, and accessories which modify a party member's stats and combat behavior. **Only 255 may be held within the inventory at a time**. While many armed items can be bought in weapon shops, some are only attainable via treasure chests or from defeating enemies. These items are explained more in Equipment.

**Goods** are a special set of items that **do not contribute to the inventory limit** for armed items or equipment, and **cannot be discarded or sold**. Most Goods simply serve as a reminders of the journey, but some are tied to gameplay. For example, the ability to **transform into a Dragoon is directly tied to whether the respective Dragoon Spirit is present in the Goods list**.

## Attack Items

- **S** targets a **single enemy**.
- **A** targets **all enemies**.
- **Multi** can be **boosted for up to 268% damage by repeatedly tapping the X button** when the on-screen graphic appears.
- **Powerful** deal more damage than those with multi even when at 268% (with the notable exception of Psychedelic Bomb and Psychedelic Bomb X).

### Item Damage Formulas

> Note: These formulas only apply when playable characters use attack items, for enemy damage formulas see Enemies.

While there are **two formulas** for calculating item damage, one for items with button mashing and one for items without it, all items use a **hidden variable for base initial damage (BID)**. Other variable multipliers can be more apparent such as the fear status or the power up state, and each is explained further below.

#### Variable Multipliers

| Item Type           | BID |
| ------------------- | --- |
| Detonate Rock       | 100 |
| All Target Multi    | 100 |
| Single Target Multi | 150 |
| All Target Powerful | 300 |
| Psyche Bomb X       | 400 |

- **Attacker Fear** — if the Attacker has the Fear Status then = (1 / 2) otherwise (1)
- **Power** — if no Power items used = (1), otherwise [(1) + (Attacker Power + Target Power)]
- **Attacker Power** — if the Attacker has power up then (1 / 2) or if power down then -(1 / 2)
- **Target Power** — if the Target has power up then -(1 / 2) or if power up then (1 / 2)
- **Field** — if the attack element neither matches nor is the opposite of special field = (1), otherwise [(1) + (Attack Element)]
- **Attack Element** — if the attack element matches the special field then (1 / 2) or if it's the opposite element then -(1 / 2)
- **Element** — if the Target Element neither matches nor is the opposite of attack element = (1), otherwise [(1) + (Target Element)]
- **Target Element** — if the Target's Element matches the attack element then -(1 / 2) or if it's the opposite element then (1 / 2)
- **Multiplier%** — refers to the % achieved through button mashing.
- **Target Fear** — if the Target has the Fear Status then (2) otherwise (1)

#### Formulas

| Applicable Items             | Formula                                                                                                                                                           |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Multi**                    | floor[floor{floor[floor{floor[floor{floor[(LV + 5) × MAT × 5 / MDF] × BID / 100} × Attacker Fear} × Power] × Field} × Element] × Multiplier% / 100] × Target Fear |
| **Powerful & Detonate Rock** | floor{floor[floor{floor[floor{floor[(LV + 5) × MAT × 5 / MDF] × BID / 100} × Target Fear × Attacker Fear] × Power} × Field] × Element}                            |

## Repeat Items

Items with **(repeat) in their description are unique in that they are returned to the inventory after each battle if they are used**. As each can only be found once, selling or discarding them renders them gone forever. **Pandemonium and Material Shield** are both found in locations that once left cannot be returned to, making them unobtainable if missed. Likewise, they can all be made unobtainable if the **point of no return in Mayfil** is taken without acquiring them. ## Trivia

The effects of **Charm Potions and Pandemonium are only ignored by four minor enemies: Knight of Sandora (Seles), Hellena Warden (with Fruegel), Guftas, and Rodriguez**. All four would not be considered minor enemies, except they are all affected by **Total Vanishing**. Possibly, the **Knights and Wardens were not intended as minor enemies but since it is normally impossible to have items or equipment that inflicts instant death when encountering them, the flag preventing instant death was not checked**.
