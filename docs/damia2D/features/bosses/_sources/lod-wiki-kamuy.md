### Abilities

Boss behavior often changes from player input as most of them use "if → then" rules. Conditions are criteria that must be met in order for a Boss to take a particular action. With multiple eligible actions, the Boss selects one at random. Below are some key terms that appear frequently, so are defined here for brevity.

- **Auto** — Found under the Conditions column, this means the Action will be used on the enemy's next turn if the other conditions are met.
- **Ignore Turn Order** — Usually paired with the Retaliate action, this means the current values which determine turn order do not change from this action being taken.

| Action              | Target | Effect                                                                                                                                                                                                           | Conditions                  |
| ------------------- | ------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------- |
| **~Do nothing**     | N/A    | Does nothing                                                                                                                                                                                                     | **Only used by Retaliate**. |
| **~Bite**           | Single | Inflicts **1x Physical damage**                                                                                                                                                                                  | -                           |
| **~Howling Blades** | Party  | Inflicts **1x Wind-elemental magic damage**                                                                                                                                                                      | -                           |
| **~Howl**           | Party  | **~99.01% chance to inflict Stun** — Only used by Retaliate. Target's A-AV reduces chance to receive status ailment. Due to the code calling for 0-100, there is a 1/101 chance (~0.99%) for the status to fail. | -                           |

\*As most abilities do not have official names, the community came up with approximate names marked with an **~** in the above table.

## Trivia

- While **not thought of as an enemy since it takes no actions and is not targetable**, **Kamuy's Tree still has a speed value and takes turns which sometimes leads to slight pauses in the battle**. Its **only significance appears to be as a set piece in the animation for Kamuy's Howling Blades**. This **trick of having untargetable enemies in the battle for graphical reasons** is **also seen in the boss fights for Lloyd (Flanvel Tower), Magician Faust (Real), all three of the Dragon Spirits, Claire, and Zieg Feld**.
