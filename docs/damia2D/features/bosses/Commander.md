## Combat flow canon

### Boss Seles flow

1. Scripted encounter Seles submap 725
2. Boss + 2 Knights of Sandora party
3. AI cycle :
   - Sword Slash (1× phys) OR Burn Out (1.2× Fire-magic) — pre-Power Up
   - When Knights of Sandora defeated → **Power Up Auto** (Ignore Turn Order Slash Twice)
   - Post-Power Up : Slash Twice (2× phys) OR Burn Out (now 1.5×)
   - HP < 51% → HP recovers (30% heal)
4. Counter 0 — freely use additions
5. Escape 0% lock-in

### Mob Marshland flow

1. Scripted encounter Marshland submap 108
2. Commander + 2 Sandora Soldier Fire party
3. AI cycle :
   - HP > 50% : Sword Slash (1× phys) — OR Stunning Hammer (100% Stun)
   - HP ≤ 50% : Multi Slash (2× phys) — OR Stunning Hammer
4. Counter 28 — cautious additions
5. Escape 100% — option flee
6. **Cannot Instant Death kill** (Immunity passive)
