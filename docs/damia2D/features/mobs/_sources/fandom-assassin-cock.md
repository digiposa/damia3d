Assassin Cock (アサシンコック, _Asashin Kokku_) is a **wind based monster found in the Forest**.

## Appearance

The Assassin Cock is a **purple bird with large claws**. The **Fowl Fighter is a recolored version of this creature**.

⚠️ **DIRECTION RECOLOR INVERSE vs wiki tier 2** : fandom dit Fowl Fighter = recolored version OF Assassin Cock (Assassin Cock = original canon). Wiki tier 2 dit Assassin Cock = recolor OF Fowl Fighter (Fowl Fighter = original). Divergence à reconcilier.

## Battle

The Assassin Cock's **high speed will often allow it to have the first strike**.

### Abilities (fandom canon names)

- **Feet Sickle** — Runs up to character and slashes with claws
- **Sound Wave** — Caws creating a multi-target attack
