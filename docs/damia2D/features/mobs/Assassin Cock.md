## Combat flow canon

1. Mob spawn random (Forest submaps 5/6/7/624/625 OU World Map roads Serdio Disc 1)
2. AI cycle :
   - HP > 50% (= HP 2-3) : ~Talon Kick (2× phys single)
   - HP ≤ 50% (= HP 1) : ~Cry (Party Non-Elemental AoE)
3. Counter mechanism : player Additions trigger counter sur 19 button presses canon
4. **HP 3** = typically defeated 1-2 hits player

### Strategy canon recommandée

- **Wind weak Earth** → Kongol axes / Earth Repeat Items 1.5× damage (mais Kongol pas encore disponible Disc 1)
- **Dart Broad Sword + initial Addition** = 1-shot HP 3
- **Avoid Counter Opportunities** : limit Lavitz Flower Storm (5 counter presses) + Rose Demon's Dance (3 presses)
- **Safe Additions** : single-press counters (Dart Volcano 1 / Lavitz Gust 1 / Meru ×3 / Haschel 1 / Albert ×2)
- **Status applicables** : Confuse / Fear / Poison / Stun (non-immune) — mais HP 3 trop bas pour utiliser tactical
- **Escape 90%** = option facile si player low level
